# Memory: features/bot-cache-analytics
Updated: now

Система аналітики кешу відстежує ефективність SSR, фіксуючи статус `HIT` або `MISS` для кожного візиту бота у таблиці `bot_visits`. Спеціальна панель 'Cache HIT' в адмін-панелі візуалізує ці дані через Recharts, відображаючи відсоткове співвідношення влучань, динаміку продуктивності (24г/7д/30д) та топ-сторінки за запитами ботів. Це дозволяє контролювати якість індексації та оцінювати економію ресурсів завдяки кешуванню.

## SSR Test Tool

Панель BotCacheAnalyticsPanel включає інструмент тестування SSR:
- Поле вводу для шляху (/sitemap, /news/us, /chapter/15 тощо)
- Вибір типу бота (Googlebot, Bingbot, GPTBot, ClaudeBot)
- Кнопка "Тест" для виконання запиту до ssr-render функції
- Кнопка "Відкрити" для перегляду SSR-HTML напряму в браузері
- Результат показує: X-Cache статус, час відповіді, розмір, наявність canonical/robots, кількість лінків
- HTML превʼю контенту який бачать боти
