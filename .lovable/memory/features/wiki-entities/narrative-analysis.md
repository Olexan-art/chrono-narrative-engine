# Memory: features/wiki-entities/narrative-analysis
Updated: now

Система наративного аналізу дозволяє адміністраторам генерувати ШІ-звіти про медіа-активність сутності за конкретний місяць. Результати зберігаються в таблиці `narrative_analyses` (багатомовна підтримка) і включають: ключові висновки (Key Takeaways), аналіз ролей пов'язаних сутностей та оцінку тональності (Sentiment) з анімованими бейджами. Користувачі бачать лише основний опис, а деталі та посилання на новини заховані під кнопкою "Читати більше". Реалізовано функцію регенерації з позначкою "Оновлено" для актуалізації даних.

## Інтеграції наративів:
1. **Hero Topic (головна сторінка)**: Якщо для ТОП-1 сутності за 24 години є наратив, під карткою показується оверлей з наративним саммарі, сентиментом і анімацією підсвітки/пульсу.
2. **Сторінка новини**: Блок "Narrative Analysis" під оригінальною новиною з саммарі головної сутності та "Читати більше" для деталей. У бічній панелі "Mentioned Entities" поряд зі сутностями показується індикатор "Є наратив" з посиланням на вікі-сторінку.
3. **Порівняння наративів**: На сторінці сутності доступно порівняння наративів за різні місяці — два дропдауни для вибору місяців, з side-by-side відображенням сентименту, саммарі та кількості тез.

## Реюзабельний компонент:
`NarrativeAnalysisBlock.tsx` — уніфікований блок відображення наративу з пропсами для compact/animated/showHeader режимів.
