# Memory: features/wiki-entities/public-ecosystem-v5
Updated: now

Розширена екосистема Wikipedia-сутностей:

1. **Алфавітний фільтр каталогу**: На сторінці /wiki додано горизонтальну панель алфавіту (UK/EN) для фільтрації сутностей за першою літерою. Фільтр зберігається в URL параметрі `letter`.

2. **Розширений граф пересічень**: Компонент EntityIntersectionGraph тепер показує до 24 сутностей (замість 15), з покращеними CSS-анімаціями (flowDown 3s, pulse з transform scale, glow ефекти).

3. **Посилання категорій на Wikipedia**: Категорії в блоці "Key Information" тепер посилаються на відповідні сторінки категорій Wikipedia (з автовизначенням мови uk/en).

4. **Автолінкування сутностей у тексті**: Новий компонент EntityLinkedContent автоматично знаходить згадки сутностей з бази у тексті extract та додає інтерактивні посилання з тултипами.

5. **Відключена автогенерація**: Автоматична генерація твітів та діалогів у RSS-пайплайні відключена (news_auto_dialogue_enabled=false, news_auto_tweets_enabled=false).

6. **Entity Graph на сторінці новини**: Сторінка NewsArticlePage тепер показує EntityIntersectionGraph головної сутності новини над блоком "OBSERVERS REACTIONS" (на десктопі у сайдбарі, на мобільних під статтею).
