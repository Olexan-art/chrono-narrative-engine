# Memory: features/wiki-entities/public-ecosystem-v3
Updated: now

Публічна екосистема Wikipedia-сутностей включає каталог (/wiki) та детальні профілі (/wiki/:slug). URL-адреси сутностей використовують friendly-slugs замість UUID (наприклад, /wiki/donald-trump-6f16b8ca). Сторінка сутності містить:

1. **Агрегований рейтинг** - комбінований показник: перегляди новин + лайки новин + лайки карикатур. Відображається у великому блоці з іконками розбивки.

2. **Hero-блок з фото** - редагування імені/опису (кнопка Edit), оновлення фото з Wikipedia (кнопка на hover).

3. **Графік динаміки** - 7-денний графік переглядів (EntityViewsChart) на основі агрегованих даних з пов'язаних новин.

4. **Topics з іконками** - теми відображаються з тематичними іконками (політика, економіка, технології тощо) замість простого тексту.

5. **Keywords** - агреговані ключові слова з усіх пов'язаних новин, відсортовані за частотою.

6. **Карикатури з лайтбоксом** - галерея карикатур з можливістю збільшення; у лайтбоксі показується новина-джерело карикатури.

7. **Новини з пагінацією** - список новин із пагінацією по 70 елементів на сторінку.

8. **Related Entities** - сутності, що згадуються в тих самих новинах, з посиланнями на їхні slug-сторінки.

Каталог сутностей (/wiki) відсортований за "останньою згадкою" (найновіші новини спочатку) та використовує slug у посиланнях. Усі сторінки оптимізовані для SEO з canonical URLs та включені до SSR-кешу.
